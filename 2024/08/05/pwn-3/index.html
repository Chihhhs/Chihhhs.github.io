bof

Hijack ret addr

rip to run call_me()

  • objdump -d ./bof

  • sub rsp,0x30 <- 48 bits in stack
  • rbp-ox30 for gets() input

  • 塞a到 0x38 <- 0x30 buf + 0x8 saved rbp
  • 然後加上 p64(call_me()的address) <- return address
  • 就會跳到call_me去執行

總結

蓋 0x30 bit 會到 rbp 所以要加 8 bit覆蓋rbp 之後才填入要去的address (使用pwntools p64())
