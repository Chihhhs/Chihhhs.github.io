Other parameter

  • 增加輸出的細部資訊
    • $ansibel-navigator run newuser.yml -m stdout -vvvv
選項v描述
-v顯示任務結果。
-vv任務結果和任務配置都會顯示
-vvv包含關於與受管主機連接的資訊
-vvvv增加了連接外掛程式相關的額外詳細程度選項，包括受管主機上用於執行腳本的用戶，以及所執行的腳本
  • $ansible-navigator run newuser.yml -m stdout --syntax-check
