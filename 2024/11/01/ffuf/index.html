FFUF

Basic

常用選項

  • -w：指定字典檔（支持多個）。
  • -u：目標 URL（用 FUZZ 表示測試位置）。
  • -v：詳細模式顯示。
  • -c：彩色輸出。
  • -mc 或 -ms：過濾或匹配 HTTP 狀態碼（如：200、403）。
  • -fc or -fs：過濾 HTTP 狀態碼或響應大小。
  • -o：將結果輸出至檔案（支持 json、html 等格式）。
  • -e：文件擴展名列表。
  • -X：HTTP 方法（GET、POST 等）。
  • -d：POST 資料。
  • -H：自定義 HTTP 標頭。
  • -recursion：啟用遞歸（針對子目錄進行掃描）。
  • -timeout：請求超時設定。
